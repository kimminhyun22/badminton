📌 프로그램 개요

배드민턴 동호회·모임용 팀 자동 배정 + 대진표 자동 생성 도구입니다. 실력 균형, 게임 수 보장, 실시간 중계, 경기 중 재배정까지 한 번에 처리합니다.

📊 명부 관리

회원 명부 저장, JSON/CSV 내보내기·불러오기 지원

⚖️ 팀 자동/수동 배정

자동 배정 후 청팀·홍팀 간 선수 수동 이동 가능

✏️ 팀명 변경

청팀/홍팀 명칭을 직접 수정, 대진표·인쇄에 반영

⏱ 예상 시간

25점·21점·15점 점수제별 예상 진행 시간 표시

💾 자동저장

대진표·점수 자동저장, 재실행 시 복원 가능

📡 실시간 중계

카톡 링크로 대진표, 팀원, 점수, MVP 현황 공유

🚀 기본 사용 순서

1

참가자 입력 영역에서 이름·급수·성별을 선택 후 + 선수 추가 버튼 클릭 (또는 Enter). 명부에서 선수를 불러올 수도 있습니다.

2

팀전을 할 경우 🔀 청/홍팀 자동 배정 버튼으로 팀을 나누거나, 선수 목록에서 팀을 직접 선택합니다. (팀전이 아니면 생략)

3

코트 수, 인당 게임 수, 혼복 횟수 등을 설정하고 🎯 대진표 생성 클릭. 버튼 한 번으로 여러 후보를 비교해 가장 좋은 대진을 자동 선택합니다.

4

각 경기의 승/패 버튼을 눌러 결과를 입력합니다. 점수는 자동 저장됩니다.

🎯 똑똑한 대진 배정

콕매치는 단순 랜덤이 아니라 다음을 함께 고려해 대진을 짭니다.

⚖️ 실력 균형

양 팀 실력 합이 비슷하도록 배정. 남녀 실력차도 반영

🔁 재대결 다양성

같은 4명이 다시 경기하거나 똑같은 팀 대결이 반복되지 않도록 분산

⏱ 대진 간격

직전 라운드 연속 출전을 최소화해 충분히 쉬도록

🎽 게임 수 보장

모든 선수가 목표 게임 수를 채우도록 배정

🟧 성비 보정

팀전 성비가 반대로 남으면 여1/남3 같은 예외 조합으로 공정성 보완

🏸 혼복 0 해석

혼복 금지가 아니라 남복·여복 우선. 필요 시 최소한의 혼복·보정경기 허용

🟧 보정경기란?

팀전에서 양팀 성비가 서로 반대로 남으면 남복·여복·혼복만으로는 전원 4게임을 정확히 맞추기 어려울 수 있습니다. 이때 콕매치는 보정경기를 만들어 일부 조합을 예외적으로 허용합니다.

예

청팀 여1/남1 vs 홍팀 남2 또는 청팀 남2 vs 홍팀 여1/남1처럼 한 경기 안의 남녀 수가 1:3 또는 3:1이 될 수 있습니다.

왜

목표 게임 수를 맞추고, 특정 선수만 3게임 또는 5게임이 되는 문제를 줄이기 위한 장치입니다.

표시

대진표에는 보정으로 따로 표시됩니다. 혼복 통계와는 별도로 집계됩니다.

주의

보정경기는 남복·여복·혼복으로 해결하기 어려울 때만 사용합니다. 품질점검의 실전 특이사항에서 개수를 확인하세요.

👥 파트너 · 게스트 지정

P

선수 목록의 P 버튼으로 두 사람을 고정 파트너로 묶을 수 있습니다. 같은 편으로 함께 경기합니다.

G

정식 회원이 아닌 손님은 G 버튼으로 게스트 표시할 수 있습니다. 이름 앞에 G 배지가 붙습니다.

🔄 경기 중 선수 변경 · 재배정

경기가 진행되는 도중에도 대진을 유연하게 조정할 수 있습니다.

✏️

개인전은 라운드별 선수 변경이 가능하지만, 팀전은 중간 변경 없이 전체 대진을 새로 만드는 방식이 안전합니다.

🎲

🎲 재배정 — 선수 변경 없이 그 라운드부터 대진만 다시 섞습니다. 결과가 마음에 안 들 때 사용하세요.

⏰

중간 투입 선수는 남은 경기 비율만큼만 게임을 배정받습니다. (예: 절반 진행 후 투입 시 4게임 중 2게임) 참가자 현황에 「중간」 배지로 표시됩니다.

🔒

중간 투입을 하면 그 이전 라운드는 🔒 변경 잠금되어 대진이 보존됩니다. 단, 승/패 입력은 계속 가능합니다.

📊 대진 품질 점검 보는 법

결과 요약의 품질 대시보드는 실력 균형, 출전 공정성, 파트너 다양성, 재대결, 휴식 간격, 일정 효율성, 설정 준수를 신호등으로 보여줍니다.

✅

양호 — 문제 없음

⚠️

주의 — 개선 여지 있으나 인원·조건상 일부 불가피할 수 있음

❌

개선 필요 — 재배정을 권장. 미달 선수, 반복 경기, 보정경기 수 등 실전 특이사항을 함께 확인하세요.

※ 인원이 적으면 연속 출전·상대 반복이 수학적으로 불가피할 수 있습니다. 대시보드는 이런 불가피한 부분을 감안해 판정합니다.

📡 실시간 중계

📡

출석부 링크를 한 번 공유하면 참석 확인, 대진표, 실시간 현황이 같은 링크로 이어집니다. 중계 시작은 새 링크를 만들지 않고 기존 링크를 라이브 화면으로 전환합니다.

🔴

중계 중에는 승/패를 입력할 때마다 자동으로 갱신됩니다. 관전자 화면의 팀 점수, MVP, 개인 순위도 함께 업데이트됩니다.

🎉

모든 경기가 끝나면 관전 화면에 경기 종료 축하 연출이 자동으로 표시됩니다. (팀전은 우승 팀, 개인전은 승률 1위)

🕓

중계 데이터는 대진 시작 후 48시간 뒤 자동으로 정리됩니다. 팀전LIVE 출석 기록은 별도로 보존되며, 중계만 정리됩니다. (즉시 끄려면 버튼을 다시 눌러 종료)

💾 자동저장 / 복원

•

대진표 생성 및 점수 입력 시 자동으로 브라우저에 저장됩니다.

•

상단의 💾 저장 / 📋 목록으로 여러 대진표를 이름 붙여 보관할 수 있습니다.

•

다른 기기와 공유하려면 ⬆️ 내보내기로 파일을 저장한 뒤 상대가 불러오면 됩니다.

•

자동 복원은 같은 기기·같은 브라우저에서만 동작합니다.
