📌 프로그램 개요

배드민턴 월례대회용 팀 자동 배정 + 대진표 자동 생성 도구입니다.

📊 구글 시트 연동

명부 탭에서 CSV로 내보내기·불러오기 지원

⚖️ 팀 자동/수동 배정

자동 배정 후 청팀·백팀 간 선수 수동 이동 가능

✏️ 팀명 변경

청팀/백팀 명칭을 직접 수정, 대진표·인쇄에 반영

⏱ 예상 시간

라운드 수 × 15분으로 총 경기 예상 시간 표시

💾 자동저장

대진표·점수 자동저장, 재실행 시 복원 가능

🖨️ A4 인쇄

깔끔한 간결 레이아웃, 청팀/백팀 색상 구분

🚀 기본 사용 순서

1

참가자 입력 영역에서 이름·급수·성별을 선택 후 + 선수 추가 버튼 클릭 (또는 Enter). 명부에서 선수를 불러올 수도 있습니다.

2

🔀 청/백팀 자동 배정 버튼으로 팀을 자동 배정하거나, 선수 목록에서 팀을 직접 선택합니다.

3

코트 수, 인당 게임 수를 설정하고 🎯 대진표 생성 클릭

4

스코어보드 팀명 입력란을 클릭해 팀 이름을 변경할 수 있습니다. (예: 블랙맘바 / 의기양양)

💾 자동저장 / 복원

•

대진표 생성 및 점수 입력 시 자동으로 브라우저에 저장됩니다.

•

파일을 다시 열면 상단에 📂 이전 대진표 불러오기 버튼이 표시됩니다.

•

같은 컴퓨터, 같은 브라우저에서 같은 HTML 파일을 열어야 복원됩니다.
