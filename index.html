대기 경기표를 자동으로 정리합니다

레슨, 늦은 합류, 먼저 귀가처럼 계속 변하는 출석 상태를 반영해 코트 수만큼 임박대기를 만들고, 예비대기는 1경기만 유지해 안정성과 다양성을 함께 지킵니다.

⚙️ 오늘운동 설정
대기중은 지금 경기 준비가 됐다는 뜻입니다. 빈 코트와 대기 가능 선수 4명 이상 조건이 맞으면 다음 대진 후보가 자동으로 표시되고, 추천을 확정하면 해당 선수는 경기중으로 바뀝니다.

예: 21:00으로 예약하면 그 시각부터 자동으로 마감 모드가 켜집니다.

상태 기준
대기중, 곧 귀가, 마지막게임이 추천 후보입니다.

추천 기준
1. 참여 가능대기중·곧 귀가·마지막게임 선수만 후보로 봅니다. 경기중·레슨중·휴식중·종료는 빠집니다.
2. 기본 균형가능하면 남복·여복·혼복 형태를 유지하고, 양팀 실력 합계 차이를 작게 맞춥니다.
3. 공정성오늘 적게 친 사람, 오래 기다린 사람, 곧 가는 사람을 우선하고 같은 파트너·상대 반복은 줄입니다.
4. 대기표코트 수만큼 임박대기를 만들고 예비대기는 1경기만 둡니다. 이후 경기 종료·시작 흐름에 맞춰 계속 보충합니다.

정렬

📌 프로그램 개요

배드민턴 동호회·모임용 팀 자동 배정 + 대진표 자동 생성 도구입니다. 실력 균형, 게임 수 보장, 실시간 중계, 경기 중 재배정까지 한 번에 처리합니다.

📊 명부 관리

회원 명부 저장, JSON/CSV 내보내기·불러오기 지원

⚖️ 팀 자동/수동 배정

자동 배정 후 청팀·홍팀 간 선수 수동 이동 가능

✏️ 팀명 변경

청팀/홍팀 명칭을 직접 수정, 대진표·인쇄에 반영

⏱ 예상 시간

25점·21점·15점 점수제별 예상 진행 시간 표시

💾 자동저장

대진표·점수 자동저장, 재실행 시 복원 가능

🗳️ 참석 링크

밴드 공지용 링크로 참석·불참·게스트 신청 자동 집계

📡 실시간 중계

카톡 링크로 대진표, 팀원, 점수, MVP 현황 공유

☀️ 오늘운동 베타

레슨·휴식·예약 대진·마감 예약을 반영해 대기 경기표 자동 생성

🚀 기본 사용 순서

1

월례대회 전에는 월례대회 참석 링크의 밴드 공지 복사를 눌러 링크를 올립니다. 회원이 직접 참석·불참·게스트를 누르면 관리자 화면에 자동 집계됩니다.

2

참석자 대진표로 가져오기를 누르면 참석 확정자와 게스트가 참가자 입력에 자동 반영됩니다. 직접 입력하거나 명부에서 불러오는 방식도 그대로 사용할 수 있습니다.

3

팀전을 할 경우 🔀 청/홍팀 자동 배정 버튼으로 팀을 나누거나, 선수 목록에서 팀을 직접 선택합니다. (팀전이 아니면 생략)

4

코트 수, 인당 게임 수, 혼복 횟수 등을 설정하고 🎯 대진표 생성 클릭. 버튼 한 번으로 여러 후보를 비교해 가장 좋은 대진을 자동 선택합니다.

5

각 경기의 승/패 버튼을 눌러 결과를 입력합니다. 점수는 자동 저장됩니다.

🎯 똑똑한 대진 배정

콕매치는 단순 랜덤이 아니라 다음을 함께 고려해 대진을 짭니다.

⚖️ 실력 균형

양 팀 실력 합이 비슷하도록 배정. 남녀 실력차도 반영

🔁 재대결 다양성

같은 4명이 다시 경기하거나 똑같은 팀 대결이 반복되지 않도록 분산

⏱ 대진 간격

직전 라운드 연속 출전을 최소화해 충분히 쉬도록

🎽 게임 수 보장

모든 선수가 목표 게임 수를 채우도록 배정

🟧 성비 보정

팀전 성비가 반대로 남으면 여1/남3 같은 예외 조합으로 공정성 보완

🏸 혼복 0 해석

혼복 금지가 아니라 남복·여복 우선. 필요 시 최소한의 혼복·보정경기 허용

🟧 보정경기란?

팀전에서 양팀 성비가 서로 반대로 남으면 남복·여복·혼복만으로는 전원 4게임을 정확히 맞추기 어려울 수 있습니다. 이때 콕매치는 보정경기를 만들어 일부 조합을 예외적으로 허용합니다.

예

청팀 여1/남1 vs 홍팀 남2 또는 청팀 남2 vs 홍팀 여1/남1처럼 한 경기 안의 남녀 수가 1:3 또는 3:1이 될 수 있습니다.

왜

목표 게임 수를 맞추고, 특정 선수만 3게임 또는 5게임이 되는 문제를 줄이기 위한 장치입니다.

표시

대진표에는 보정으로 따로 표시됩니다. 혼복 통계와는 별도로 집계됩니다.

주의

보정경기는 남복·여복·혼복으로 해결하기 어려울 때만 사용합니다. 품질점검의 실전 특이사항에서 개수를 확인하세요.

👥 파트너 · 게스트 · 예약 대진

P

선수 목록의 P 버튼으로 두 사람을 고정 파트너로 묶을 수 있습니다. 같은 편으로 함께 경기합니다.

G

정식 회원이 아닌 손님은 G 버튼으로 게스트 표시할 수 있습니다. 이름 앞에 G 배지가 붙습니다.

예약

오늘운동 예약 대진은 특정 2명을 같은 편으로 넣고 상대를 자동 배정하거나, 특정 4명의 맞대결을 1회성으로 대기표에 우선 반영합니다.

마감

마감 예약에 시각을 입력하면 해당 시각부터 자동으로 마감 모드가 켜져 새 경기 생성을 멈춥니다. 마감 5분 전부터는 완료 후 자동 승계도 멈춰 실수로 새 경기가 시작되는 일을 줄입니다.

🔄 경기 중 선수 변경 · 재배정

경기가 진행되는 도중에도 대진을 유연하게 조정할 수 있습니다.

✏️

각 라운드 제목 옆 ✏️ 선수 변경 — 늦게 온 선수를 추가하거나 먼저 간 선수를 빼고, 그 라운드부터 다시 배정합니다. 팀전은 새 선수 중간투입을 막고 기존 선수 제외만 허용합니다.

🎲

🎲 재배정 — 선수 변경 없이 그 라운드부터 대진만 다시 섞습니다. 결과가 마음에 안 들 때 사용하세요.

⏰

중간 투입 선수는 남은 경기 비율만큼만 게임을 배정받습니다. (예: 절반 진행 후 투입 시 4게임 중 2게임) 참가자 현황에 「중간」 배지로 표시됩니다.

🔒

중간 투입을 하면 그 이전 라운드는 🔒 변경 잠금되어 대진이 보존됩니다. 단, 승/패 입력은 계속 가능합니다.

📊 대진 품질 점검 보는 법

결과 요약의 품질 대시보드는 실력 균형, 출전 공정성, 파트너 다양성, 재대결, 휴식 간격, 일정 효율성, 설정 준수를 신호등으로 보여줍니다.

✅

양호 — 문제 없음

⚠️

주의 — 개선 여지 있으나 인원·조건상 일부 불가피할 수 있음

❌

개선 필요 — 재배정을 권장. 미달 선수, 반복 경기, 보정경기 수 등 실전 특이사항을 함께 확인하세요.

※ 인원이 적으면 연속 출전·상대 반복이 수학적으로 불가피할 수 있습니다. 대시보드는 이런 불가피한 부분을 감안해 판정합니다.

📡 실시간 중계

📡

결과 화면의 📡 실시간 중계 버튼을 누르면 관전용 링크가 만들어집니다. 단톡방에 링크를 공유하면 회원들이 휴대폰으로 대진표와 팀원을 바로 확인할 수 있어요.

🔴

중계 중에는 승/패를 입력할 때마다 자동으로 갱신됩니다. 관전자 화면의 팀 점수, MVP, 개인 순위도 함께 업데이트됩니다.

🎉

모든 경기가 끝나면 관전 화면에 경기 종료 축하 연출이 자동으로 표시됩니다. (팀전은 우승 팀, 개인전은 승률 1위)

🕓

중계 데이터는 48시간 후 자동으로 정리됩니다. 별도로 종료하지 않아도 며칠 뒤 사라지니 신경 쓰지 않아도 됩니다. (즉시 끄려면 버튼을 다시 눌러 종료)

☀️ 오늘운동 베타

월례대회처럼 전체 대진표를 한 번에 만드는 것이 아니라, 평소 운동일에 대기 상태를 보고 다음 경기들을 미리 정리하는 모드입니다.

1

오늘운동 메뉴에서 대진표 명단 가져오기로 대진표 탭의 참가자를 가져오거나, 명부 가져오기로 클럽 회원을 선택해 불러옵니다. 게스트는 G 버튼을 켜고 추가합니다.

2

레슨중·휴식중·곧 귀가·마지막게임·종료 상태를 바꿔 현재 운동 가능한 사람만 남깁니다. 추천 후보는 대기중, 곧 귀가, 마지막게임 선수입니다.

3

특정 두 사람을 같은 편으로 넣어달라는 요청이 있으면 선수 카드의 묶기를 누른 뒤 상대를 선택합니다. 묶음 요청은 같은 경기·같은 편 조건으로 반영됩니다.

4

예약 대진은 특정 2명을 같은 편으로 넣고 상대를 자동 배정하거나, 특정 4명의 맞대결을 1회성으로 예약합니다. 이미 확정된 임박대기는 유지하고 새 대기표를 채울 때 우선 반영됩니다.

5

임박대기는 코트 수만큼 만들고, 예비대기는 1경기만 유지합니다. 코트 수만큼 한 바퀴가 돌면 곧 시작할 1순위와 준비 알림을 보낸 대기표를 지키고, 나머지 대기표는 오래 기다린 사람 중심으로 다시 섞습니다.

6

임박대기에는 다음 코트의 바로 시작/종료임박/예상 종료 상태와 준비 알림, 경기 시작 버튼이 표시됩니다. 준비 알림은 카톡에 보낼 문구를 공유하거나 복사하는 기능입니다.

7

경기가 빨리 끝나면 조기 종료, 예상 시간이 지난 뒤에는 경기 종료를 누릅니다. 방금 끝난 선수들은 대기자로 돌아가고 다음 대기표가 보충됩니다. 자동 시작을 꺼둔 기본 운영에서는 빈 코트 카드에서 관리자가 1순위 임박대기를 직접 시작합니다. 남은 시간이 5분 이하이면 빨간색 종료임박으로 표시됩니다.

8

마감 예약에 시각을 입력하면 그 시각부터 자동으로 마감 모드가 켜져 새 경기 생성을 멈춥니다. 마감 5분 전부터는 완료 후 자동 승계를 멈춥니다. 예: 19:00 시작, 21:00 운영 마감 후 자율게임 전환.

9

조건 충족 시 자동 시작을 켜면 빈 코트와 대기 경기표 조건이 맞을 때 자동으로 진행중 경기를 만듭니다. 실전에서는 기본 OFF로 두고 관리자가 확인한 뒤 수동 시작하는 운영을 권장합니다.
