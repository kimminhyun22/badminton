# 콕매치

배드민턴 동호회·모임용 팀 배정 및 자동 대진표 생성 PWA입니다.

- 배포 주소: https://kimminhyun22.github.io/badminton/
- 실시간 중계 화면: `view.html`
- 현재 버전: `1.10.84`
- Firebase 프로젝트: `kokmatch-23b31`

## 주요 기능

- 참가자 직접 입력 및 명부 관리
- 청팀/홍팀 자동 배정과 수동 조정
- 인당 게임 수 기준 자동 대진 생성
- 남복/여복/혼복 우선 배정
- 팀전 성비 불균형 시 보정경기 자동 배정
- 대진 품질 점검
- 승패 입력과 팀 스코어 집계
- 카카오톡 공유용 실시간 중계 링크
- 자동저장, 이름 붙여 저장, JSON 백업
- 모바일 이미지 저장 및 A4 인쇄

## 운영 메모

- `혼복 0`은 혼복 금지가 아니라 남복/여복 우선이라는 뜻입니다.
- 팀전에서 성비가 서로 반대로 남으면 `보정` 경기가 생길 수 있습니다.
- 팀전에서는 경기 중 새 선수 추가를 막고, 기존 선수 제외 후 재배정만 허용합니다.
- GitHub Pages 배포는 `main` 브랜치 push 후 보통 1~3분 안에 반영됩니다.

## 개발 메모

코드 수정 시 `index.html`의 버전 3곳을 함께 올립니다.

- `APP VERSION` 주석
- `<meta name="app-version">`
- `const APP_VERSION`

유사 프로그램 개발을 위한 상세 요약은 [DEVELOPMENT_SUMMARY.md](DEVELOPMENT_SUMMARY.md)를 참고하세요.
