## 3. 오늘운동 기능 요약

### 운영 목표

- 3~4코트, 20~30명 안팎이 수시로 들어오고 나가는 평일 운동을 관리합니다.
- 레슨중, 휴식중, 먼저 귀가, 마지막게임 요청을 반영합니다.
- 카톡방에 출석 링크를 보내 회원이 직접 상태를 전송할 수 있습니다.
- 전체 대진표를 미리 확정하지 않고, 코트가 빌 때마다 다음 경기를 짧게 확정합니다.

### 상태

- `대기중`: 바로 경기 가능
- `경기중`: 진행중 경기 카드에서만 완료/취소 가능
- `레슨중`, `휴식중`: 자동 대기표 제외
- `곧 귀가`: 추천 후보이며 우선순위 상승
- `마지막게임`: 추천 후보이며 경기 완료 후 `종료`
- `종료`: 자동 대기표 제외

### 대기표 정책

- 목표 대기표 수는 `코트 수 + 1`입니다.
- 예: 3코트면 임박대기 3경기 + 예비대기 1경기.
- 단, 참여 가능 선수가 부족하면 만들 수 있는 경기 수만 유지합니다.
- 임박대기는 수정잠금입니다.
- 예비대기는 1경기만 유지하고 선수 수정 가능하게 둡니다.
- 예약 대진은 임박/예비 위치와 관계없이 수정잠금입니다.

### 경기 흐름

1. `dailyEnsureQueue()`가 대기표를 채웁니다.
2. `dailyStartQueueItem()`이 1경기를 진행중으로 보냅니다.
3. `dailyCompleteMatch()`가 게임 수와 파트너/상대 이력을 기록합니다.
4. 자동배정 ON이면 완료된 코트에 1순위 임박대기를 즉시 승계합니다.
5. 자동배정 OFF이면 빈 코트 카드에서 관리자가 1순위 임박대기를 직접 시작합니다.
6. 한 웨이브가 돌면 곧 시작할 1순위와 준비 알림을 보낸 대기표를 보존하고 나머지는 다시 섞습니다.

### 예약 대진

- 같은 편 예약: 2명을 같은 팀으로 묶고 상대는 자동 선택.
- 4명 맞대결 예약: A팀 2명, B팀 2명을 지정.
- 예약자가 경기중/레슨중/휴식중이면 대기표 반영 보류.
- 예약 대진이 시작되면 예약 목록에서 제거.
- v1.10.106에서 예약 대기표 직접 수정/자동 재배정도 함수 레벨에서 차단했습니다.
- v1.10.107에서 준비 알림을 보낸 대기표는 웨이브 재섞기 때 보존하도록 보완했습니다.

### 마감 예약

- 지정 시각부터 마감 모드가 켜지면 새 경기 자동 시작과 대기표 생성을 멈춥니다.
- 진행중 경기만 마무리합니다.
- 실전에서는 수동 마감보다 마감 예약을 미리 설정하는 쪽이 안전합니다.
- v1.10.107부터 마감 5분 전에는 완료 후 자동 승계를 멈춰 마감 직전 실수로 새 경기가 시작되는 것을 줄입니다.

## 4. 현재 검증 결과

### 단위 기능 검증

Node VM 기반으로 오늘운동 함수 경로를 직접 실행했습니다.

- 총 24개 체크 통과
- 게스트 추가 및 중복 차단
- 대진표 명단 가져오기 중복 제외
- 레슨/휴식/종료 선수 대기표 제외
- 마지막게임 후보 포함 및 완료 후 종료 처리
- 대기표 중복 선수 없음
- 묶음 요청 분리 방지
- 같은 편 예약 생성 및 우선 반영
- 예약 대기표 수정잠금
- 예약 경기 시작 시 예약 목록 제거
- 경기 시작 시 4명 경기중 전환
- 경기 완료 시 게임 수/파트너/상대 이력 기록
- 경기 취소 시 이전 상태 복원
- 마감 모드 대기표 비움 및 자동 시작 차단
- 지난 마감 예약 즉시 마감 처리
- 타이머 `soon`, `due` 상태
- 저장/복원 후 선수/예약 유지
- 대기표 미반영 예약 선수 삭제 시 관련 예약 제거
- 예약 대기표 직접 수정/자동 재배정 함수 차단
- 누적 60경기 상태에서 대기표 생성 84ms
- 자동배정 OFF에서 완료 후 자동 승계 차단
- 자동배정 ON에서 완료 후 자동 승계 유지
- 날짜 경계에서 이전 기록 새로 시작 처리
- 취소 후 웨이브 카운트 복구
- 준비 알림 보낸 대기표 재섞기 보존

### 현실 스트레스 테스트

가정:

- 3코트
- 34명
- 19:00 시작
- 21:00 운영 마감
- 12~15명 레슨
- 레슨 전후 10분 준비/정비 포함
- 선수 도착/퇴장 시간이 제각각
- 예약 대진 일부 포함

결과:

- 대기표 정합성 오류: 0
- 마감 예약이 먼저 발동된 경우 마감 이후 새 경기 시작: 0
- 평균 완료 경기: 약 21경기
- 예약 대진은 선수 상태가 맞을 때 반영됨

주의할 경계:

- 자동배정 OFF가 기본 권장값입니다. 이 경우 경기 완료 후 다음 경기는 자동 시작되지 않고, 빈 코트 카드에서 관리자가 직접 시작합니다.
- 자동배정 ON을 쓰는 경우에도 마감 예약 5분 전부터는 완료 후 자동 승계를 멈춥니다.
- 실전 운영은 `마감 예약`을 미리 걸어두는 것이 안전합니다.
